This folder will contain all of the external Arduino libraries neccessary for programming your robot if you choose to use the provided PCB and recommended components.
Libraries such as "Dabble" (used for HM-10 bluetooth communication with the Dabble application) or "AltSoftSerial" (for renaming your bluetooth device) can be
added through the Arduino IDE (tools > library manager > type in library name in search bar and click install). Any libraries provided in this folder must be downloaded 
to your Arduino libraries folder. This is most often found at within your computers documents folder: "Documents > Arduino > Libraries". Be sure to 
unzip any files that are downloaded as a .zip file.
